# 사군자 기질 검사

매화·난초·국화·대나무 중 당신의 기질을 알아보는 성향 테스트 웹앱입니다.

## 실행 방법

서버 없이 바로 실행할 수 있습니다.

- **메인**: `index.html`을 브라우저에서 열기
- **검사**: 메인에서 "검사 시작하기" 클릭 시 `test.html`로 이동
- 로컬 서버 사용: `npx serve .` 후 브라우저에서 접속

## 화면 구성

| 화면 | 파일 | 설명 |
|------|------|------|
| 메인 | `index.html` | 앱 소개 + 검사 시작하기 버튼 |
| 검사 | `test.html` | 문항 진행, 선택 현황, 이전/다음/검사 중단 |
| 결과 | `test.html` | 기질 결과, 성향 설명, 점수 비율, 다시 검사하기 |

## 검사 방식

- **문항**: 기질별 7문항 × 4종류 = **총 28문항** (매화 → 난초 → 국화 → 대나무 순)
- **척도**: 5점 리커트 척도 (1: 전혀 그렇지 않다 ~ 5: 매우 그렇다)
- **진행**: 이전/다음으로 자유 이동, 미응답 문항 가능
- **선택 현황**: 문항별 선택값 표시, 미응답 개수, 클릭 시 해당 문항으로 이동
- **결과**: 4가지 기질 점수 합산, 최고점 기질(또는 혼합형) 결과 표시

## 주요 기능

- **다크 모드**: 우측 상단 토글 (설정은 localStorage에 저장)
- **검사 중단**: 검사 중 메인(`index.html`)으로 돌아가기
- **결과 보기**: 마지막 문항에서 "결과 보기" 클릭 (미응답 시 확인 후 진행)
- **다시 검사하기**: 결과 화면에서 `test.html` 처음부터 재진행
- **반응형**: 모바일·PC 대응, 검사 화면은 스크롤 없이 비율 맞춤

## 파일 구조

```
tfgp/
├── index.html      # 메인 (검사 시작 링크)
├── test.html       # 검사 + 결과
├── css/
│   └── style.css   # 공통 스타일 (라이트/다크, 반응형)
├── js/
│   ├── common.js   # 다크 모드 토글 (메인·검사 공용)
│   └── test.js     # 검사 로직 (문항, 척도, 점수, 결과)
└── README.md
```

- `app.js`는 구 버전(단일 페이지)용이며, 현재는 사용하지 않습니다.

## 기술 스택

- HTML5, CSS3 (모바일 우선 반응형), Vanilla JavaScript
- 서버·DB·로그인 없음 (프론트 전용)

## 브라우저최신 Chrome, Safari, Edge, Firefox (모바일 포함) 권장.
